# E-commerce Sales & Customer Behavior Analysis

Проект присвячений аналізу даних електронної комерції. Основна мета — оптимізація прибутковості через сегментацію клієнтів та виявлення паттернів поведінки.

## Стек технологій
- **Python:** Data cleaning, ETL.
- **SQL:** Аналіз транзакцій, агрегації.
- **Git:** Контроль версій, структура проекту.

## Структура проекту
- `/data` — набір даних.
- `/scripts` — автоматизація очищення даних.
- `/sql` — SQL-запити для аналітики.
- `/notebooks` — дослідження та візуалізація.

*Деталі аналізу будуть додані після завершення етапів розробки.*
